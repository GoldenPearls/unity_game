# 유니티 게임 : 반려동물을 찾아서..
학교 전공심화 과제로 낸 게임으로 2~3주 정도의 제작 기간으로 유니티를 이용하였고 강의를 보고 클론 코딩하여 조금 변형하여 제작하게 되었다. 3d 쿼터뷰 슈팅게임
